## STEPS FOR SIMULATION:
1. Open the LTspice software.
2. Go to File â†’ **New Schematic**.
3. Click the **Component** symbol to browse and select components.
4. Find the **diode** or **diodes** from the component list and place them.
5. Click **OK** and place the component.
6. Search for a **resistor** from the component menu or from the toolbar icons.
7. Place the resistors.
8. Search for a **capacitor** from the component menu or from the toolbar icons.
9. Place the capacitors.
10. Use **Ctrl + R** to rotate the components.
11. Use the **Wire** symbol to connect the components as per the circuit diagram.
12. Right-click each component to assign values.
13. Use the **Run** button to start the simulation.
14. Set the step time and click **OK**.
15. The **Output** dialog box will open.
16. In the output dialog box, right-click and select **Add Traces**.
17. Node voltages and currents for each component will be displayed.
18. Select the required node voltages (e.g., **V(n001)**) to display the waveforms.
19. To get the voltage across a component, the trace name should include both nodes (example: **V(n001,n002)**).
